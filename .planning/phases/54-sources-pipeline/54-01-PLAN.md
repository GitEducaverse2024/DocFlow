<objective>
Create the Sources Pipeline — a simplified 3-phase wizard (Fuentes → Procesar → Indexar RAG) that replaces the complex 8-step pipeline for the common use case of adding and indexing sources.

The wizard is triggered from the Entry Modal "Nuevas Fuentes" button (Phase 53) and renders within the CatBrain detail page.

Key technical requirements:
1. **RAG append**: When RAG is already active (`rag_enabled=1`), use `/api/catbrains/[id]/rag/append` — do NOT recreate the collection.
2. **Content fallback**: The append endpoint already has filename fallback (`"Archivo: {name}"`), but Phase 3 UI must handle the 400 error gracefully and show per-source failure details.
3. **Progress bar**: Phase 3 polls `/rag/status` and shows a real progress bar with `chunksProcessed/chunksTotal` percentage and elapsed time.

Output: 1 new component + 2 modified files + i18n keys in both languages.
</objective>

   **Append path** (`ragInfo.enabled`):
   ```typescript
   // Get source IDs that are pending append
   // Mark all current sources as pending first
   const pendingIds = sources.filter(s => sourceModes[s.id] !== 'exclude').map(s => s.id);

   // Mark them as pending_append in DB
   for (const id of pendingIds) {
     await fetch(`/api/catbrains/${catbrainId}/sources/${id}`, {
       method: 'PATCH',
       headers: { 'Content-Type': 'application/json' },
       body: JSON.stringify({ is_pending_append: 1 }),
     });
   }

   // Call append endpoint
   setIsIndexing(true);
   const res = await fetch(`/api/catbrains/${catbrainId}/rag/append`, {
     method: 'POST',
     headers: { 'Content-Type': 'application/json' },
     body: JSON.stringify({ sourceIds: pendingIds }),
   });

   if (res.ok) {
     const data = await res.json();
     setIndexResult({ vectorsAdded: data.vectors_added, sourcesProcessed: data.sources_processed });
     setIndexComplete(true);
   } else {
     const err = await res.json();
     setIndexError(err.error || 'Error en indexación');
     // If response has failures array, show per-source details
     if (err.failures) {
       setIndexResult({ vectorsAdded: 0, sourcesProcessed: 0, failures: err.failures });
     }
   }
   setIsIndexing(false);
   ```

   ```typescript
   setIsIndexing(true);
   const res = await fetch(`/api/catbrains/${catbrainId}/rag/create`, {

     body: JSON.stringify({

     if (data.status === 'running') {
       const processed = data.chunksProcessed || 0;
       const total = data.chunksTotal || 1;
       const percent = Math.round((processed / total) * 100);
       setIndexProgress({ processed, total, percent });
     } else if (data.status === 'completed') {
       clearInterval(pollInterval);
       setIndexComplete(true);
       setIndexResult({ vectorsAdded: data.chunksCount || 0, sourcesProcessed: sources.length });
       setIsIndexing(false);
     } else if (data.status === 'error') {
       clearInterval(pollInterval);
       setIndexError(data.error || 'Error desconocido');
       setIsIndexing(false);
     }
     ```
