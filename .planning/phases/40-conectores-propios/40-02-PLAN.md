API contract:
- GET /api/catbrains/:id/connectors -> CatBrainConnector[]

- PATCH /api/catbrains/:id/connectors/:connId { name?, type?, config?, description?, is_active? } -> CatBrainConnector
- DELETE /api/catbrains/:id/connectors/:connId -> { success: true }
- POST /api/catbrains/:id/connectors/:connId/test -> { success, test_status, message, duration_ms }
