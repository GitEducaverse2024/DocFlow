<task type="auto">
  <name>Task 1: Create catbrain-connector-executor service</name>
  <files>app/src/lib/services/catbrain-connector-executor.ts</files>
  <action>
Create `app/src/lib/services/catbrain-connector-executor.ts` with:

1. **Main function**: `executeCatBrainConnectors(catbrainId: string, query: string, mode: 'connector' | 'both'): Promise<ConnectorResult[]>`

   This function:
   - Queries DB: `SELECT * FROM catbrain_connectors WHERE catbrain_id = ? AND is_active = 1`
   - If no active connectors, return empty array
   - For each active connector, execute in parallel (Promise.allSettled) with individual 15s timeout
   - Return array of results

2. **ConnectorResult interface** (export):
   ```typescript
   export interface ConnectorResult {
     connector_id: string;
     connector_name: string;
     connector_type: string;
     success: boolean;
     data: unknown;
     error?: string;
     duration_ms: number;
   }
   ```

3. **Per-type execution logic** (internal function `executeConnector`):
   - `n8n_webhook`: POST to config.url with body { query, source: 'catbrain', catbrain_id }. Return response JSON.
   - `http_api`: Use config.method (default GET). If POST/PUT/PATCH, substitute `{{query}}` in config.body_template with the actual query. Add config.headers. Return response JSON or text.
   - `mcp_server`: This is the CatBrain-to-CatBrain case. Send JSON-RPC request to config.url:
     ```json
     { "jsonrpc": "2.0", "id": 1, "method": "tools/call", "params": { "name": "search_knowledge", "arguments": { "query": query } } }
     ```
     Parse the MCP response and extract content. This enables querying another CatBrain's RAG via its MCP endpoint.
   - `email`: Skip during automatic execution (email is fire-and-forget notification, not a data connector). Return { skipped: true }.

4. **Error handling**: Each connector execution is wrapped in try-catch. Failures do NOT block other connectors. Log errors via logger.error('connectors', ...).

5. **Logging**: Log start/end of connector execution batch with logger.info('connectors', 'Executing catbrain connectors', { catbrainId, count, mode }).

Import db from '@/lib/db', logger from '@/lib/logger'. Use AbortController for timeout per connector.
