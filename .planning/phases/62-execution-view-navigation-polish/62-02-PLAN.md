# Plan 62-02: Read-Only Canvas Execution Modal

## Goal
Add a read-only React Flow modal that shows live canvas execution progress with colored nodes, opened from the "Ver canvas en tiempo real" link on canvas step cards.

## Context
- Plan 62-01 adds the canvas step card with progress bar and a placeholder "Ver canvas en tiempo real" link
- Canvas editor already uses `@xyflow/react` with custom node types in `app/src/components/canvas/nodes/`
- Canvas runs store `node_states` (JSON with per-node status/output) and `execution_order` in the DB
- The existing canvas editor (`app/src/components/canvas/canvas-editor.tsx`) is a full editor — we need a simplified read-only viewer
- Canvas `flow_data` contains the nodes array with positions and edge connections

## Tasks

### Task 1: Create API endpoint for canvas run data

**File**: `app/src/app/api/canvas-runs/[id]/route.ts` (NEW)

GET endpoint that returns canvas_run data + canvas flow_data for the modal:

```typescript
import { NextResponse } from 'next/server';
import db from '@/lib/db';

export const dynamic = 'force-dynamic';

export async function GET(request: Request, { params }: { params: { id: string } }) {
  const run = db.prepare(`
    SELECT cr.id, cr.canvas_id, cr.status, cr.node_states, cr.current_node_id,
           cr.execution_order, cr.total_tokens, cr.total_duration,
           c.name as canvas_name, c.flow_data
    FROM canvas_runs cr
    JOIN canvases c ON c.id = cr.canvas_id
    WHERE cr.id = ?
  `).get(params.id) as Record<string, unknown> | undefined;

  if (!run) return NextResponse.json({ error: 'Canvas run not found' }, { status: 404 });

  return NextResponse.json({
    id: run.id,
    canvas_name: run.canvas_name,
    status: run.status,
    node_states: JSON.parse(run.node_states as string || '{}'),
    current_node_id: run.current_node_id,
    execution_order: JSON.parse(run.execution_order as string || '[]'),
    flow_data: JSON.parse(run.flow_data as string || '{"nodes":[],"edges":[]}'),
    total_tokens: run.total_tokens,
    total_duration: run.total_duration,
  });
}
```

### Task 2: Create CanvasLiveModal component

**File**: `app/src/components/canvas/canvas-live-modal.tsx` (NEW)

Dynamic import component (SSR disabled) with read-only React Flow:

1. **Props**: `{ canvasRunId: string; isOpen: boolean; onClose: () => void }`
2. **State**: `canvasRun` data fetched from `/api/canvas-runs/[id]`
3. **Polling**: When `isOpen && canvasRun.status === 'running'`, poll every 2s
4. **React Flow setup**:
   - Load nodes from `flow_data.nodes`, edges from `flow_data.edges`
   - Map each node's visual style based on `node_states[nodeId].status`:
     - `pending`: `bg-zinc-800 border-zinc-700 text-zinc-400`
     - `running`: `bg-violet-900/30 border-violet-500 text-violet-300 animate-pulse`
     - `completed`: `bg-emerald-900/20 border-emerald-500 text-emerald-300`
     - `failed`: `bg-red-900/20 border-red-500 text-red-300`
     - `skipped`: `bg-zinc-800 border-zinc-700 text-zinc-500 opacity-50`
   - Use a generic node type (simple colored box with label + status icon) rather than importing the full editor node types
   - Configure: `nodesDraggable={false}`, `nodesConnectable={false}`, `elementsSelectable={false}`, `panOnDrag={true}`, `zoomOnScroll={true}`, `fitView={true}`

5. **Node component** (inline or separate):
   ```tsx
   function LiveNode({ data }: { data: { label: string; type: string; status: string } }) {
     const statusStyles = {
       pending: 'bg-zinc-800 border-zinc-700 text-zinc-400',
       running: 'bg-violet-900/30 border-violet-500 text-violet-300 animate-pulse',
       completed: 'bg-emerald-900/20 border-emerald-500 text-emerald-300',
       failed: 'bg-red-900/20 border-red-500 text-red-300',
       waiting: 'bg-amber-900/20 border-amber-500 text-amber-300',
       skipped: 'bg-zinc-800 border-zinc-700 text-zinc-500 opacity-50',
     };
     const style = statusStyles[data.status] || statusStyles.pending;
     return (
       <div className={`px-4 py-2 rounded-lg border-2 ${style} min-w-[120px] text-center`}>
         <div className="text-xs text-zinc-500 uppercase">{data.type}</div>
         <div className="text-sm font-medium truncate">{data.label}</div>
       </div>
     );
   }
   ```

6. **Modal layout**:
   ```tsx
   <Dialog open={isOpen} onOpenChange={onClose}>
     <DialogContent className="max-w-4xl h-[70vh] bg-zinc-950 border-zinc-800 p-0">
       <DialogHeader className="px-6 pt-6 pb-2">
         <DialogTitle className="flex items-center gap-2">
           <Workflow className="w-5 h-5 text-violet-400" />
           {canvasRun?.canvas_name}
           <Badge className={statusClass}>{canvasRun?.status}</Badge>
         </DialogTitle>
       </DialogHeader>
       <div className="flex-1 min-h-0">
         <ReactFlowProvider>
           <ReactFlow
             nodes={coloredNodes}
             edges={flow_data.edges}
             nodeTypes={{ default: LiveNode }}
             nodesDraggable={false}
             nodesConnectable={false}
             elementsSelectable={false}
             fitView
           >
             <Background />
             <Controls showInteractive={false} />
           </ReactFlow>
         </ReactFlowProvider>
       </div>
     </DialogContent>
   </Dialog>
   ```
