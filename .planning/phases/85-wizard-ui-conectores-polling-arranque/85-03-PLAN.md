Updated register function should be:

```typescript
export async function register() {
  if (process['env']['NEXT_RUNTIME'] === 'nodejs') {
    const { taskScheduler } = await import('@/lib/services/task-scheduler');
    taskScheduler.start();

    // Start Drive polling daemon (skip in test env)
    if (process['env']['NODE_ENV'] !== 'test') {
      try {
        const { DrivePollingService } = await import('@/lib/services/drive-polling');
        DrivePollingService.getInstance().start();
      } catch (err) {
        console.error('[instrumentation] Failed to start DrivePollingService:', err);
      }
    }
  }
}
```

Key points:
- Use `process['env']['NODE_ENV']` with bracket notation per project convention
- Use `process['env']['NEXT_RUNTIME']` with bracket notation (already in place)
- Wrap in try/catch so a missing or broken drive-polling module does not crash the entire app startup
- Dynamic import (`await import(...)`) so the module is only loaded at runtime, not bundled
- The `.start()` method internally checks for active sync_jobs; if none exist it simply returns without scheduling anything
