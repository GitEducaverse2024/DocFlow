1. In `db.ts`, add after the canvas tables block (before the "Mark stuck" section):
   ```sql
   CREATE TABLE IF NOT EXISTS notifications (
     id TEXT PRIMARY KEY,
     type TEXT NOT NULL,
     title TEXT NOT NULL,
     message TEXT,
     severity TEXT DEFAULT 'info',
     link TEXT,
     read INTEGER DEFAULT 0,
     created_at TEXT DEFAULT (datetime('now'))
   );
   ```
   Also add a startup cleanup: delete notifications older than 30 days (same pattern as log rotation):
   ```typescript
   try {
     db.prepare("DELETE FROM notifications WHERE created_at < datetime('now', '-30 days')").run();
   } catch { /* table may not exist on first run */ }
   ```

2. Create `app/src/lib/services/notifications.ts` with:
   - Types: `NotificationType = 'process' | 'rag' | 'task' | 'canvas' | 'connector' | 'system'` and `NotificationSeverity = 'success' | 'info' | 'warning' | 'error'`
   - Interface `CreateNotificationParams { type, title, message, severity, link? }`
   - Interface `Notification { id, type, title, message, severity, link, read, created_at }`
   - `createNotification(params)`: INSERT with crypto.randomUUID() for id. Wrap the entire function body in try-catch that logs errors via `logger.error('notifications', ...)` but never throws (notification creation must never crash the caller). Use the project's `generateId()` pattern if available, otherwise inline `crypto.randomUUID()`.
   - `getNotifications(opts: { limit?, offset?, type?, severity? })`: SELECT with optional WHERE filters, ORDER BY created_at DESC, returns `{ notifications: Notification[], total: number }` (total = count without limit/offset for pagination)
   - `getUnreadCount()`: SELECT COUNT(*) WHERE read = 0
   - `markAsRead(id: string)`: UPDATE SET read = 1 WHERE id = ?
   - `markAllAsRead()`: UPDATE SET read = 1 WHERE read = 0
   - Import db from `@/lib/db` and logger from `@/lib/logger`

3. Create four API routes, ALL with `export const dynamic = 'force-dynamic';`:

   **GET /api/notifications/route.ts**: Parse searchParams: `limit` (default 20), `offset` (default 0), `type` (optional), `severity` (optional). Call `getNotifications(...)`. Return JSON `{ notifications, total, limit, offset }`.

   **GET /api/notifications/count/route.ts**: Call `getUnreadCount()`. Return JSON `{ count }`.

   **PATCH /api/notifications/[id]/read/route.ts**: Extract `id` from params. Call `markAsRead(id)`. Return JSON `{ success: true }`.

   **POST /api/notifications/read-all/route.ts**: Call `markAllAsRead()`. Return JSON `{ success: true }`.

   All routes: import from `@/lib/services/notifications`, wrap in try-catch returning 500 on error, use `logger.info('notifications', ...)` for logging. Use `process['env']` bracket notation if any env vars are needed.
