# Plan 61-04: Import Endpoint + Bundle Importer Service

## Goal
Create the import endpoint that accepts a ZIP bundle upload and idempotently imports all resources.

## Context
- Plan 61-01 defines the bundle structure: manifest.json, config/ (task.json, canvases/, agents/, skills/)
- Resources map to tables: cat_paws (agents), skills, canvases, tasks + task_steps
- Idempotent: match existing resources by name — skip if exists, create if not
- `unzip` binary is available in the Docker image (Dockerfile installs it)
- Need to remap IDs: exported IDs → new generated IDs for all cross-references

## Tasks

**Implementation flow:**

1. **Validate manifest**: Read `manifest.json`, check `bundle_version` and `task` fields exist. Throw descriptive error if invalid.

2. **Import agents** (config/agents/*.json):
   - Read each JSON file in `config/agents/`
   - Check `cat_paws` table for existing row with same `name`
   - If exists: use existing ID, mark `created: false`
   - If not: INSERT into `cat_paws` with new UUID, copy fields (name, description, avatar_emoji, avatar_color, system_prompt, tone, mode, model, temperature, max_tokens, processing_instructions, output_format)
   - Build `idMap.agents: Record<oldId, newId>`

3. **Import skills** (config/skills/*.json):
   - Same pattern — match by name in `skills` table
   - INSERT new with fields (name, description, category, tags, instructions, output_template, constraints, source='imported')
   - Build `idMap.skills: Record<oldId, newId>`
   - Re-create `worker_skills` entries using remapped agent + skill IDs

4. **Import canvases** (config/canvases/*.json):
   - Match by name in `canvases` table
   - INSERT new with fields (name, description, emoji, mode, flow_data, tags, status='active')
   - Build `idMap.canvases: Record<oldId, newId>`

5. **Import task** (config/task.json):
   - Always create new task with new UUID (tasks are never skipped)
   - Remap step references: `agent_id` → `idMap.agents[step.agent_id]`, `canvas_id` → `idMap.canvases[step.canvas_id]`
   - INSERT task with all fields, status='draft'
   - INSERT task_steps with remapped references and new UUIDs

6. **Return ImportResult** with counts of created vs skipped

### Task 2: Create import API route

**File**: `app/src/app/api/tasks/import/route.ts`

```typescript
import { NextResponse } from 'next/server';
import fs from 'fs';
import path from 'path';
import os from 'os';
import { importBundle } from '@/lib/services/bundle-importer';

export const dynamic = 'force-dynamic';

export async function POST(request: Request) {
  try {
    const formData = await request.formData();
    const file = formData.get('file') as File;

    if (!file || !file.name.endsWith('.zip')) {
      return NextResponse.json({ error: 'ZIP file required' }, { status: 400 });
    }

    // Save uploaded ZIP to temp dir
    const tmpDir = path.join(os.tmpdir(), `docflow-import-${Date.now()}`);
    fs.mkdirSync(tmpDir, { recursive: true });
    const zipPath = path.join(tmpDir, file.name);
    const buffer = Buffer.from(await file.arrayBuffer());
    fs.writeFileSync(zipPath, buffer);

    // Extract ZIP (unzip is available in Docker image)
    const { execSync } = require('child_process');
    execSync(`unzip -o "${zipPath}" -d "${tmpDir}"`);

    // Find the bundle directory (first directory in extracted)
    const entries = fs.readdirSync(tmpDir).filter(
      e => fs.statSync(path.join(tmpDir, e)).isDirectory()
    );
    const bundleDir = entries.length > 0 ? path.join(tmpDir, entries[0]) : tmpDir;

    const result = await importBundle(bundleDir);

    // Cleanup temp dir
    fs.rmSync(tmpDir, { recursive: true, force: true });

    return NextResponse.json(result, { status: 201 });
  } catch (error: any) {
    console.error('Import error:', error);
    return NextResponse.json(
      { error: error.message || 'Failed to import bundle' },
      { status: 500 }
    );
  }
}
```
