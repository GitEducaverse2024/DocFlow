# Plan 61-03: Docker Templates + Install Scripts + Runner HTML

## Goal
Create template generators for all non-config bundle files and integrate them into the bundle generator ZIP.

## Context
- Plan 61-01 creates the ZIP with config/ directory only
- This plan adds docker/, install/, and runner/ directories
- Templates are TypeScript functions returning string content, parameterized by manifest data
- Docker compose must use `image:` (not `build:`), with pinned version tags
- The app version is read from package.json at generation time

## Tasks

### Task 1: Create docker-compose + install script templates

**File**: `app/src/lib/export-templates/docker-compose.yml.ts`

Generate docker-compose.yml with conditional services. All images use pinned version tags:
- `docflow/app:v15.0` (read version from package.json)
- `ghcr.io/berriai/litellm:1.63.14` (pinned)
- `qdrant/qdrant:v1.12.6` (pinned)
- `ollama/ollama:0.5.7` (pinned)

```typescript
export function generateDockerCompose(
  services: string[],
  credentials: string[],
  appVersion: string  // e.g., '15.0' from package.json
): string {
  const envVars = credentials.map(c => `      - ${c}=\${${c}}`).join('\n');

  let yaml = `version: '3.8'\n\nservices:\n  docflow:\n    image: docflow/app:v${appVersion}\n    ports:\n      - "3500:3000"\n    environment:\n      - NODE_ENV=production\n      - DATA_DIR=/app/data\n${envVars}\n    volumes:\n      - docflow-data:/app/data\n    restart: unless-stopped\n`;

  if (services.includes('litellm')) {
    yaml += `\n  litellm:\n    image: ghcr.io/berriai/litellm:1.63.14\n    ports:\n      - "4000:4000"\n    environment:\n      - LITELLM_MASTER_KEY=\${LITELLM_API_KEY}\n    restart: unless-stopped\n`;
  }
  if (services.includes('qdrant')) {
    yaml += `\n  qdrant:\n    image: qdrant/qdrant:v1.12.6\n    ports:\n      - "6333:6333"\n    volumes:\n      - qdrant-data:/qdrant/storage\n    restart: unless-stopped\n`;
  }
  if (services.includes('ollama')) {
    yaml += `\n  ollama:\n    image: ollama/ollama:0.5.7\n    ports:\n      - "11434:11434"\n    volumes:\n      - ollama-data:/root/.ollama\n    restart: unless-stopped\n`;
  }

  yaml += `\nvolumes:\n  docflow-data:\n`;
  if (services.includes('qdrant')) yaml += `  qdrant-data:\n`;
  if (services.includes('ollama')) yaml += `  ollama-data:\n`;

  return yaml;
}
```

**File**: `app/src/lib/export-templates/install-scripts.ts`

Single file with three exports: `generateInstallSh()`, `generateInstallPs1()`, `generateSetupWizard(credentials)`.

- `generateInstallSh()`: Bash script — checks docker, checks docker compose, runs `node install/setup-wizard.js`, `docker compose pull`, `docker compose up -d`
- `generateInstallPs1()`: PowerShell equivalent with `Write-Host`, `Get-Command`, `Set-Location docker`
- `generateSetupWizard(credentials)`: Node.js script using `readline` — prompts for each credential, writes `docker/.env` file

(Full content as specified in the previous plan iteration — all three functions in one file since they are closely related install tooling.)

### Task 2: Create runner HTML template

**File**: `app/src/lib/export-templates/runner-html.ts`

`generateRunnerHtml(taskId, taskName)` returns a standalone HTML page with:
- Dark theme (#0a0a0a background, violet accent #7c3aed) matching DocFlow brand
- "Ejecutar tarea" button → POST `/api/tasks/{id}/execute`
- `setInterval(pollStatus, 2000)` → GET `/api/tasks/{id}/status`
- Step progress display with colored left borders (pending=gray, running=amber, completed=green, failed=red)
- Progress bar (completed steps / total steps)
- Result display area with "Descargar resultado" button (creates text blob download)
- Error display area

### Task 3: Integrate templates into bundle-generator.ts + tests

**File**: `app/src/lib/services/bundle-generator.ts`

Import all template generators. After the config/ files, append to archive:

```typescript
import { generateDockerCompose } from '@/lib/export-templates/docker-compose.yml';
import { generateInstallSh, generateInstallPs1, generateSetupWizard } from '@/lib/export-templates/install-scripts';
import { generateRunnerHtml } from '@/lib/export-templates/runner-html';

// Read app version from package.json
const pkg = JSON.parse(fs.readFileSync(path.join(process.cwd(), 'package.json'), 'utf-8'));
const appVersion = pkg.version || '0.1.0';

// docker/
archive.append(generateDockerCompose(manifest.docker_services, manifest.credentials_needed, appVersion),
  { name: `${prefix}/docker/docker-compose.yml` });

// install/
archive.append(generateInstallSh(), { name: `${prefix}/install/install.sh`, mode: 0o755 });
archive.append(generateInstallPs1(), { name: `${prefix}/install/install.ps1` });
archive.append(generateSetupWizard(manifest.credentials_needed),
  { name: `${prefix}/install/setup-wizard.js`, mode: 0o755 });

// runner/
archive.append(generateRunnerHtml(task.id, task.name),
  { name: `${prefix}/runner/index.html` });
```

**File**: `app/src/lib/export-templates/export-templates.test.ts`

```typescript
describe('export templates', () => {
  it('docker-compose includes only required services with pinned versions', () => {
    const yaml = generateDockerCompose(['docflow', 'litellm', 'qdrant'], ['LITELLM_API_KEY'], '15.0');
    expect(yaml).toContain('qdrant/qdrant:v1.12.6');
    expect(yaml).not.toContain('ollama');
    expect(yaml).toContain('image:');
    expect(yaml).not.toContain('build:');
    expect(yaml).toContain('docflow/app:v15.0');
  });

  it('install.sh checks for Docker', () => {
    const sh = generateInstallSh();
    expect(sh).toContain('#!/bin/bash');
    expect(sh).toContain('docker');
  });

  it('runner HTML contains task ID and polls at 2s interval', () => {
    const html = generateRunnerHtml('abc-123', 'Mi Tarea');
    expect(html).toContain('abc-123');
    expect(html).toContain('pollStatus');
    expect(html).toContain('2000');
  });

  it('setup wizard prompts for all credentials', () => {
    const js = generateSetupWizard(['OPENAI_API_KEY', 'LITELLM_API_KEY']);
    expect(js).toContain('OPENAI_API_KEY');
    expect(js).toContain('LITELLM_API_KEY');
  });
});
```
