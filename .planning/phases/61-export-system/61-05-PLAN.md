# Plan 61-05: Export Section UI + i18n

## Goal
Add a collapsible export section to the task detail page showing resource summary, bundle generation, and previous bundle management.

## Context
- Task detail page at `app/tasks/[id]/page.tsx` uses client-side React with fetch
- API routes from plans 61-01 and 61-02 provide: POST export, GET exports, GET download, DELETE bundle
- i18n uses `next-intl` with `useTranslations()` and keys in `messages/es.json` + `messages/en.json`
- UI uses shadcn/ui components (Button, Badge) + lucide-react icons
- Dark theme with violet accent (#7c3aed)

### Task 2: Create ExportSection component in task detail page

**File**: `app/src/app/tasks/[id]/page.tsx`

Add `Package` and `Trash2` to the lucide-react imports.

Create `ExportSection` component (inline in the same file):

**State:** `isOpen`, `bundles[]`, `generating`, `loaded`

**Behavior:**
- Collapsed by default; clicking header toggles open/close
- On first open: fetches `GET /api/tasks/${taskId}/exports` to load bundle list
- "Generar bundle" button: calls `POST /api/tasks/${taskId}/export`, then refreshes bundle list
- Resource summary grid (3 columns: agents, skills, canvases) from latest bundle's parsed manifest
- Required services shown as Badge components
- Bundle list: each row shows name, date, download button (opens `/api/tasks/${taskId}/exports/${bundleId}/download` in new tab), delete button (with confirm)
- Toast notifications via `sonner` on generate/delete success/error

**Layout structure:**
```
┌─────────────────────────────────────────────┐
│ 📦 Exportar                          [▼/▶] │
├─────────────────────────────────────────────┤
│ Description text (text-zinc-400)            │
│                                             │
│ ┌─────────┬─────────┬─────────┐             │
│ │ 3       │ 2       │ 1       │  (counts)   │
│ │ Agentes │ Skills  │ Canvas  │             │
│ └─────────┴─────────┴─────────┘             │
│                                             │
│ Servicios: [docflow] [litellm] [qdrant]     │
│                                             │
│ [🔮 Generar bundle]                         │
│                                             │
│ Bundles anteriores:                         │
│ ┌───────────────────────────────────────┐   │
│ │ mi-tarea-1234.zip   ⬇️  🗑️          │   │
│ │ 2026-03-21 14:30                      │   │
│ └───────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

Place `<ExportSection taskId={task.id} />` after the existing result/output section in the page.
