1. In `app/src/lib/utils.ts`, add and export a `generateId()` function that generates UUID v4 strings using Math.random (NOT crypto.randomUUID which requires HTTPS):
```typescript
export function generateId(): string {
  return 'xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx'.replace(/[xy]/g, (c) => {
    const r = (Math.random() * 16) | 0;
    return (c === 'x' ? r : (r & 0x3) | 0x8).toString(16);
  });
}
```
