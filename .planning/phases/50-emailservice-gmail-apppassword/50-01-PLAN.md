```typescript
import nodemailer from 'nodemailer';
import type { Transporter } from 'nodemailer';

import { logger } from '@/lib/logger';
```

**createTransporter(config: GmailConfig): Transporter**
- If `config.auth_mode === 'app_password'`:
  - Decrypt `config.app_password_encrypted` using `decrypt()`
  - Strip spaces from decrypted password (App Passwords may be pasted with spaces)
  - If `config.account_type === 'personal'`: use `service: 'gmail'` (smtp.gmail.com:587)
  - If `config.account_type === 'workspace'`: use `host: 'smtp-relay.gmail.com', port: 587, secure: false` with STARTTLS
  - Auth: `{ user: config.user, pass: decryptedPassword }`
- If `config.auth_mode === 'oauth2'`:
  - Import `google` from `googleapis`
  - Create OAuth2Client with decrypted client_id, client_secret
  - Set credentials with decrypted refresh_token
  - Get accessToken via `oauth2Client.getAccessToken()`
  - Create transporter with `service: 'gmail'`, auth type 'OAuth2', user, clientId, clientSecret, refreshToken, accessToken
- Never log credentials. Log only `config.user` and `config.account_type`.

**testConnection(config: GmailConfig): Promise<{ ok: boolean; error?: string }>**
- Create transporter, call `transporter.verify()`
- Wrap in `withRetry` with `maxAttempts: 2` and `shouldRetry` that returns false for auth errors (strings containing 'auth', '535', '534', 'invalid', 'credentials')
- Return `{ ok: true }` on success, `{ ok: false, error: translateError(err) }` on failure
- Add helper `translateError(err: Error): string` that maps common Nodemailer errors to Spanish:
  - ECONNREFUSED -> "No se pudo conectar al servidor SMTP"
  - 535/534 -> "Credenciales invalidas. Verifica tu App Password"
  - ETIMEDOUT -> "Tiempo de espera agotado al conectar con Gmail"
  - self signed -> "Error de certificado SSL"
  - default -> original error message

**sendEmail(config: GmailConfig, payload: EmailPayload): Promise<{ ok: boolean; messageId?: string; error?: string }>**
- Validate payload: `to` and `subject` are required, at least one of `html_body` or `text_body`
- Create transporter
- Build mail options: `from` as `"${config.from_name || 'DoCatFlow'} <${config.user}>"`, `to` (join if array), `subject`, `html` from html_body, `text` from text_body, `replyTo` if present
- Wrap `transporter.sendMail(mailOptions)` in `withRetry` with `maxAttempts: 2`, shouldRetry skips auth errors
- Return `{ ok: true, messageId: info.messageId }` or `{ ok: false, error: translateError(err) }`
- Log success/failure (never log credentials, only log to/subject/messageId)

Export all three functions and the translateError helper.
