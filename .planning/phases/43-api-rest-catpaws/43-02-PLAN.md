**1. `/api/cat-paws/[id]/relations/route.ts`** (API-06) — GET only:
- Load 3 relation sets with JOINs (same queries as 43-01 detail, but standalone endpoint):
  - catbrains: `SELECT cpc.*, c.name as catbrain_name FROM cat_paw_catbrains cpc LEFT JOIN catbrains c ON c.id = cpc.catbrain_id WHERE cpc.paw_id = ?`
  - connectors: `SELECT cpc.*, cn.name as connector_name, cn.type as connector_type FROM cat_paw_connectors cpc LEFT JOIN connectors cn ON cn.id = cpc.connector_id WHERE cpc.paw_id = ?`
  - agents: `SELECT cpa.*, cp2.name as target_name, cp2.avatar_emoji as target_emoji FROM cat_paw_agents cpa LEFT JOIN cat_paws cp2 ON cp2.id = cpa.target_paw_id WHERE cpa.paw_id = ?`
- Return `{ catbrains, connectors, agents }`

**2. `/api/cat-paws/[id]/catbrains/route.ts`** (API-07) — POST only:
- Body: `{ catbrain_id, query_mode?, priority? }`
- Validate: catbrain_id required (400), check catbrain exists in catbrains table (404 "CatBrain not found")
- Defaults: query_mode = 'rag', priority = 0
- INSERT INTO cat_paw_catbrains (paw_id, catbrain_id, query_mode, priority, created_at)
- Handle UNIQUE constraint violation: catch error, if message includes "UNIQUE", return 409 "Ya vinculado"
- Return inserted row with 201

**3. `/api/cat-paws/[id]/catbrains/[catbrainId]/route.ts`** (API-08) — DELETE only:
- Params: `{ params: Promise<{ id: string; catbrainId: string }> }`
- DELETE FROM cat_paw_catbrains WHERE paw_id = ? AND catbrain_id = ?
- Check `changes` property: if 0, return 404 "Vinculo no encontrado"
- Return `{ success: true }`

**4. `/api/cat-paws/[id]/connectors/route.ts`** (API-09) — GET + POST:
- GET: list connectors for this paw with JOIN to connectors table for name/type
- POST body: `{ connector_id, usage_hint?, is_active? }`
- Validate: connector_id required (400), check connector exists in connectors table (404)
- Defaults: is_active = 1
- INSERT INTO cat_paw_connectors (paw_id, connector_id, usage_hint, is_active, created_at)
- Handle UNIQUE constraint: 409
- Return inserted row with 201

**5. `/api/cat-paws/[id]/connectors/[connectorId]/route.ts`** — DELETE:
- DELETE FROM cat_paw_connectors WHERE paw_id = ? AND connector_id = ?
- If changes === 0, return 404
- Return `{ success: true }`

**6. `/api/cat-paws/[id]/agents/route.ts`** (API-10) — POST:
- Body: `{ target_paw_id, relationship? }`
- Validate: target_paw_id required (400), check target paw exists (404), prevent self-link (paw_id === target_paw_id → 400 "No se puede vincular consigo mismo")
- Default: relationship = 'collaborator'
- Validate relationship is one of: collaborator, delegate, supervisor
- INSERT INTO cat_paw_agents (paw_id, target_paw_id, relationship, created_at)
- Handle UNIQUE constraint: 409
- Return inserted row with 201

**7. `/api/cat-paws/[id]/agents/[targetPawId]/route.ts`** — DELETE:
- DELETE FROM cat_paw_agents WHERE paw_id = ? AND target_paw_id = ?
- If changes === 0, return 404
- Return `{ success: true }`

**`app/src/app/api/agents/route.ts`** — Replace entire file:
```typescript
import { NextResponse } from 'next/server';
export const dynamic = 'force-dynamic';

export async function GET(request: Request) {
  const url = new URL(request.url);
  const target = new URL('/api/cat-paws', url.origin);
  target.search = url.search; // preserve query params
  return NextResponse.redirect(target.toString(), 301);
}

export async function POST(request: Request) {
  const url = new URL(request.url);
  return NextResponse.redirect(new URL('/api/cat-paws', url.origin).toString(), 308);
}
```

Use 301 for GET (permanent redirect, cacheable) and 308 for POST (permanent redirect that preserves method — unlike 301 which may change POST to GET).
