```typescript
interface CascadeSectionProps {
  index: number;           // 0-based section index
  title: string;           // Section title from i18n
  isCompleted: boolean;    // Section has been completed
  isActive: boolean;       // Currently expanded/active
  isLocked: boolean;       // Cannot interact (sections beyond current)
  summary: string;         // One-line summary when collapsed + completed
  onToggle: () => void;    // Click to expand/collapse
  children: React.ReactNode;
}
```

Render:
- Outer div: `border rounded-lg` with `border-violet-500/40` when active, `border-zinc-800` otherwise, `opacity-50 pointer-events-none` when locked
- Header row: clickable, shows numbered circle (1-5), title, summary text (only when completed AND collapsed), chevron icon
- Circle states: completed = `bg-emerald-500/20 text-emerald-400` with Check icon, active = `bg-violet-500 text-white` with number, locked = `bg-zinc-800 text-zinc-500` with number
- Content area: only shown when `isActive || (isCompleted && user clicked to reopen)` -- the parent controls this via `isActive`
- Animate content with simple conditional render (no framer-motion needed)
