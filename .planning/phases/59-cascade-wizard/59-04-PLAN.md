5. **Fork group reconstruction algorithm** (explicit):
   ```
   a. Collect all steps where fork_group is not null
   b. Group by fork_group value -> Map<string, TaskStep[]>
   c. For each fork_group:
      i.   Find the step with type='fork' -> this is the fork entry step
      ii.  Find the step with type='join' -> this is the join step
      iii. Remaining steps: sort by branch_index ASC
      iv.  Group remaining steps by branch_index -> branches
      v.   For each branch_index group:
           - Extract branch_label from the first step in that group
           - Map steps to PipelineStep objects
           - Create ForkBranch: { label: branch_label, steps: PipelineStep[] }
      vi.  Assemble: { forkGroupId: fork_group, branches: ForkBranch[], joinStep: PipelineStep }
   d. Insert reconstructed fork groups into pipelineSteps at the correct order_index position
   ```

6. Set all sections as completed, expand section 4 (Revisar) by default

**B. Edit mode save flow** (definitive approach -- no deliberation):

The save flow for edit mode is: PATCH task fields, DELETE all existing steps, POST new steps in order.

```typescript
async function saveTaskEditMode(launch: boolean) {
  // 1. PATCH task fields
  await fetch(`/api/tasks/${editId}`, {
    method: 'PATCH',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      name: taskName,
      description: taskDescription,
      expected_output: expectedOutput,
      linked_projects: JSON.stringify(selectedProjects),
      execution_mode: executionMode,
      execution_count: executionMode === 'variable' ? executionCount : 1,
      schedule_config: executionMode === 'scheduled' ? scheduleConfig : null,
      status: 'ready',
    }),
  });

  // 2. DELETE all existing steps
  const existingSteps = await fetch(`/api/tasks/${editId}/steps`).then(r => r.json());
  await Promise.all(
    existingSteps.map((step: { id: string }) =>
      fetch(`/api/tasks/${editId}/steps/${step.id}`, { method: 'DELETE' })
    )
  );

  // 3. POST new steps in order
  for (let i = 0; i < flattenedSteps.length; i++) {
    await fetch(`/api/tasks/${editId}/steps`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ ...flattenedSteps[i], order_index: i }),
    });
  }

  // 4. If launching, execute
  if (launch) {
    await fetch(`/api/tasks/${editId}/execute`, { method: 'POST' });
  }
}
```
