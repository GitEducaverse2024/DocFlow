  <action>
    **use-log-viewer.ts** — Custom hook managing log data and filter state.

    **State:**
    - `entries: LogEntry[]` — current log entries from API
    - `level: string` — 'all' | 'info' | 'warn' | 'error' (default 'all')
    - `source: string` — 'all' | LogSource value (default 'all')
    - `search: string` — text search (default '')
    - `loading: boolean`
    - `autoRefresh: boolean` — default true, controls polling

    **Interface:**
    ```typescript
    interface LogEntry { ts: string; level: string; source: string; message: string; metadata?: Record<string, unknown> }
    ```

    **Functions:**
    - `fetchLogs()` — Build query string from level/source/search (omit 'all' values). GET /api/system/logs?{params}. Set entries.
    - `downloadLogs()` — Open `/api/system/logs/download` in new window (`window.open`). This triggers the browser download.
    - `setLevel`, `setSource`, `setSearch` — state setters

    **Effects:**
    - On mount + when level/source/search change: call fetchLogs().
    - Polling: useEffect with setInterval 3000ms when autoRefresh is true. Clear on cleanup or when autoRefresh changes.
    - Debounce search: use a 500ms debounce on search changes before triggering fetchLogs (useRef + setTimeout pattern).

    **Return:** `{ entries, level, source, search, loading, autoRefresh, setLevel, setSource, setSearch, setAutoRefresh, fetchLogs, downloadLogs }`

    **log-filters.tsx** — Client component receiving filter state and setters as props.

    Props: `{ level, source, search, autoRefresh, onLevelChange, onSourceChange, onSearchChange, onAutoRefreshChange, onDownload }`

    Layout: horizontal flex row with gap-3, wrapping on mobile.
    - **Level select**: `<select>` with options: Todos, Info, Warn, Error. Values: 'all', 'info', 'warn', 'error'. Styled: zinc-900 bg, zinc-300 text, rounded-lg, border-zinc-700.
    - **Source select**: `<select>` with options: Todas, Processing, Chat, RAG, CatBot, Tasks, Canvas, Connectors, System, Agents, Workers, Skills, Settings, Notifications. Values: 'all' + all LogSource values.
    - **Search input**: text input with Search icon (lucide), placeholder "Buscar en logs...", zinc-900 bg.
    - **Auto-refresh toggle**: small button/checkbox "Auto-refrescar" that toggles autoRefresh.
    - **Download button**: "Descargar logs" button with Download icon, zinc-700 bg. Calls onDownload.
    - All text in Spanish.
  </action>
