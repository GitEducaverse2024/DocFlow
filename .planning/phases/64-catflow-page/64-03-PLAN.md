Filter type:
```typescript
type FilterKey = 'all' | 'active' | 'scheduled' | 'listening' | 'draft';
```

Filter logic:
- **Todos**: all tasks
- **Activos**: status in ['ready', 'running', 'paused', 'completed'] (not draft, not failed)
- **Programados**: execution_mode === 'scheduled'
- **En escucha**: listen_mode === 1
- **Borradores**: status === 'draft'

Counts computed from full task list.

Data fetching:
```typescript
const fetchData = useCallback(async () => {
  const [tasksRes, templatesRes] = await Promise.all([
    fetch('/api/tasks'),
    fetch('/api/tasks/templates'),
  ]);
  if (tasksRes.ok) {
    const data = await tasksRes.json();
    setTasks(data);
  }
  if (templatesRes.ok) setTemplates(await templatesRes.json());
  setLoading(false);
}, []);
```

Note: GET /api/tasks already returns all task columns via `SELECT *`, so listen_mode, execution_mode, etc. are already in the response — we just need to type them properly in CatFlowTask.

Handler functions (inside CatFlowPageContent):

**handleToggleActive**:
```typescript
const handleToggleActive = async (id: string, currentStatus: string) => {
  const newStatus = currentStatus === 'draft' ? 'ready' : 'draft';
  try {
    const res = await fetch(`/api/tasks/${id}`, {
      method: 'PATCH',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ status: newStatus }),
    });
    if (!res.ok) throw new Error();
    setTasks(prev => prev.map(t => t.id === id ? { ...t, status: newStatus } : t));
    toast.success(t(newStatus === 'ready' ? 'toasts.activated' : 'toasts.deactivated'));
  } catch {
    toast.error(t('toasts.toggleError'));
  }
};
```

**handleToggleListenMode**:
```typescript
const handleToggleListenMode = async (id: string, currentMode: number) => {
  const newMode = currentMode === 1 ? 0 : 1;
  try {
    const res = await fetch(`/api/tasks/${id}`, {
      method: 'PATCH',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ listen_mode: newMode }),
    });
    if (!res.ok) throw new Error();
    setTasks(prev => prev.map(t => t.id === id ? { ...t, listen_mode: newMode } : t));
    toast.success(t(newMode === 1 ? 'toasts.listenEnabled' : 'toasts.listenDisabled'));
  } catch {
    toast.error(t('toasts.toggleError'));
  }
};
```
