  <files>app/src/lib/services/stream-utils.ts</files>
  <action>
Create `app/src/lib/services/stream-utils.ts` with the following exports:

1. **`StreamOptions` interface:** `{ model: string; messages: Array<{ role: string; content: string; tool_calls?: unknown[]; tool_call_id?: string }>; max_tokens?: number; tools?: unknown[] }`

2. **`StreamCallbacks` interface:** `{ onToken: (token: string) => void; onToolCall?: (toolCall: { id: string; type: string; function: { name: string; arguments: string } }) => void; onDone: (usage?: { prompt_tokens: number; completion_tokens: number; total_tokens: number }) => void; onError: (error: Error) => void; }`

3. **`streamLiteLLM(options: StreamOptions, callbacks: StreamCallbacks): Promise<void>`** — Core function that:
   - Reads `process['env']['LITELLM_URL']` (default `http://localhost:4000`) and `process['env']['LITELLM_API_KEY']` (default `sk-antigravity-gateway`) using bracket notation
   - Calls `${litellmUrl}/v1/chat/completions` with `stream: true` and `stream_options: { include_usage: true }`
   - If response is not ok, throws with status + body text
   - Reads response body via `getReader()` + `TextDecoder` with `{ stream: true }`
   - Maintains a string buffer, splits by `\n`, keeps last incomplete line for next iteration
   - For each `data: ` line: skips `[DONE]`, parses JSON, extracts `delta.content` -> `onToken`, handles `delta.tool_calls` with index-based accumulator (new tool call when `tc.id` exists, append arguments otherwise), captures `parsed.usage` from final chunk
   - On `finish_reason === 'tool_calls'`: calls `onToolCall` for each accumulated tool call, then clears the accumulator
   - Calls `onDone(usage)` after the read loop completes normally
   - Calls `onError(error)` in catch block
   - Does NOT use `withRetry` (per project decision: withRetry does NOT apply to streaming LLM calls)
   - Logs stream start with `logger.info` and errors with `logger.error`, source = 'system'

4. **`sseHeaders` constant:** `{ 'Content-Type': 'text/event-stream', 'Cache-Control': 'no-cache, no-transform', 'Connection': 'keep-alive', 'X-Accel-Buffering': 'no' }`

5. **`createSSEStream(handler: (send: (event: string, data: unknown) => void, close: () => void) => void): ReadableStream`** — Utility that:
   - Creates a `TextEncoder`
   - Returns `new ReadableStream({ start(controller) { ... } })`
   - Defines `send(event, data)` as `controller.enqueue(encoder.encode(\`event: ${event}\ndata: ${JSON.stringify(data)}\n\n\`))`
   - Defines `close()` as `controller.close()`
   - Calls `handler(send, close)` inside start — the handler runs an async IIFE internally, the ReadableStream returns immediately

Import `logger` from `@/lib/logger`.

4. **If `useStream` is truthy**, after building `contextChunks` (lines ~55-58 in current code), instead of the non-streaming LiteLLM fetch:
   - Read env vars: `litellmUrl`, `litellmKey`, `chatModel` (same bracket notation as current code)
   - Record `chatStartTime = Date.now()`
   - Build the SSE stream using `createSSEStream`:
     ```
     const sseStream = createSSEStream((send, close) => {
       (async () => {
         try {
           let fullContent = '';
           send('start', { timestamp: Date.now() });

           await streamLiteLLM(
             { model: chatModel, messages: [systemMsg, userMsg] },
             {
               onToken: (token) => {
                 fullContent += token;
                 send('token', { content: token });
               },
               onDone: (usage) => {
                 logUsage({ event_type: 'chat', project_id: projectId, model: chatModel, input_tokens: usage?.prompt_tokens || 0, output_tokens: usage?.completion_tokens || 0, total_tokens: usage?.total_tokens || 0, duration_ms: Date.now() - chatStartTime, status: 'success' });
                 send('done', { usage, sources: results });
                 close();
               },
               onError: (error) => {
                 logger.error('chat', 'Error en streaming', { projectId, error: error.message });
                 send('error', { message: error.message });
                 close();
               },
             }
           );
         } catch (error) {
           send('error', { message: (error as Error).message });
           close();
         }
       })();
     });
     return new Response(sseStream, { headers: sseHeaders });
     ```
