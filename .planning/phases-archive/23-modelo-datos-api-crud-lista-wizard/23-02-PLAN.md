**File 1: `app/src/app/api/canvas/[id]/validate/route.ts`** (DATA-09)

`export const dynamic = 'force-dynamic';`

POST handler — validates the canvas DAG and returns errors array:
1. Load canvas flow_data by id from DB. Return 404 if not found.
2. Parse flow_data JSON (default to `{ nodes: [], edges: [] }` if null).
3. Run 4 validation checks, accumulating errors in a string array:
   - **START exists:** At least one node with type 'start'. Error: "El canvas necesita un nodo Inicio (START)"
   - **OUTPUT exists:** At least one node with type 'output'. Error: "El canvas necesita al menos un nodo Salida (OUTPUT)"
   - **No orphans:** Every non-start node must have at least one incoming edge (appear as a target in edges). Error: "Nodos sin conexion de entrada: {list of orphan labels or ids}"
   - **No cycles:** Implement DFS-based cycle detection. Build adjacency list from edges. For each node, run DFS tracking visited + recursion stack. If a node is revisited while in the recursion stack, there is a cycle. Error: "El canvas contiene ciclos — solo se permiten DAGs"
4. Return `{ valid: errors.length === 0, errors }`

The cycle detection function should be a helper function within the file (~15-20 lines):
```typescript
function detectCycle(nodes: Array<{id: string}>, edges: Array<{source: string; target: string}>): boolean {
  const adj = new Map<string, string[]>();
  nodes.forEach(n => adj.set(n.id, []));
  edges.forEach(e => adj.get(e.source)?.push(e.target));

  const visited = new Set<string>();
  const inStack = new Set<string>();

  function dfs(nodeId: string): boolean {
    visited.add(nodeId);
    inStack.add(nodeId);
    for (const neighbor of (adj.get(nodeId) || [])) {
      if (inStack.has(neighbor)) return true;
      if (!visited.has(neighbor) && dfs(neighbor)) return true;
    }
    inStack.delete(nodeId);
    return false;
  }

  for (const node of nodes) {
    if (!visited.has(node.id) && dfs(node.id)) return true;
  }
  return false;
}
```
