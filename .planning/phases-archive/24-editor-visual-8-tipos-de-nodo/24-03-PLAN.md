**Part A: Undo/Redo (EDIT-09)**

In canvas-editor.tsx (CanvasShell):

1. Add state for undo/redo history:
   ```typescript
   const [past, setPast] = useState<{ nodes: Node[]; edges: Edge[] }[]>([]);
   const [future, setFuture] = useState<{ nodes: Node[]; edges: Edge[] }[]>([]);
   ```

2. Create `takeSnapshot` callback (useCallback, deps: [nodes, edges]):
   - Push current { nodes, edges } to past (keep last 30 snapshots)
   - Clear future array
   - Call: `setPast(prev => [...prev.slice(-30), { nodes: [...nodes], edges: [...edges] }]); setFuture([]);`

3. Create `undo` callback:
   - If past is empty, return
   - Save current state to future: `setFuture(f => [{ nodes: [...nodes], edges: [...edges] }, ...f])`
   - Pop last from past: `const prev = past[past.length - 1]`
   - `setPast(p => p.slice(0, -1))`
   - Restore: `setNodes(prev.nodes); setEdges(prev.edges);`

4. Create `redo` callback:
   - If future is empty, return
   - Save current state to past: `setPast(p => [...p, { nodes: [...nodes], edges: [...edges] }])`
   - Pop first from future: `const next = future[0]`
   - `setFuture(f => f.slice(1))`
   - Restore: `setNodes(next.nodes); setEdges(next.edges);`

5. Capture snapshots at key moments — call `takeSnapshot()` BEFORE:
   - Adding a node (in onDrop handler, before setNodes)
   - Connecting nodes (in onConnect, before setEdges)
   - Deleting nodes/edges (add onNodesDelete and onEdgesDelete callbacks: takeSnapshot then allow default behavior)

6. Add keyboard listener via useEffect:
   ```typescript
   useEffect(() => {
     const handleKeyDown = (e: KeyboardEvent) => {
       if ((e.ctrlKey || e.metaKey) && e.key === 'z' && !e.shiftKey) {
         e.preventDefault();
         undo();
       }
       if ((e.ctrlKey || e.metaKey) && e.key === 'z' && e.shiftKey) {
         e.preventDefault();
         redo();
       }
       // Also Ctrl+Y as redo alternative
       if ((e.ctrlKey || e.metaKey) && e.key === 'y') {
         e.preventDefault();
         redo();
       }
     };
     document.addEventListener('keydown', handleKeyDown);
     return () => document.removeEventListener('keydown', handleKeyDown);
   }, [undo, redo]);
   ```

7. Pass `onUndo`, `onRedo`, `canUndo: past.length > 0`, `canRedo: future.length > 0` to toolbar.

8. In canvas-toolbar.tsx, add Undo/Redo buttons (Undo2 and Redo2 icons from lucide-react) next to the save status, disabled when history is empty.
