**Key functions:**
- `fetchConnectors()` — GET /api/connectors, set state
- `handleCreate()` — Open sheet with empty form, set editingConnector to null
- `handleEdit(connector)` — Open sheet, populate form from connector
- `handleSave()` — POST (create) or PATCH (update) based on editingConnector
- `handleDelete(id)` — Confirm dialog then DELETE
- `handleTest(id)` — POST /api/connectors/{id}/test, show result toast
- `handleShowLogs(connector)` — GET /api/connectors/{id}/logs, open dialog
- `handleUseTemplate(template)` — Pre-fill form from template and open sheet
