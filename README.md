# DocFlow

DocFlow es una aplicación web local que gestiona proyectos de documentación inteligente. El usuario crea proyectos, sube fuentes heterogéneas (documentos, URLs, YouTube, notas), conecta agentes IA para procesar esa documentación, e indexa los resultados en una base vectorial para consulta RAG.

## Instalación

1. Clona el repositorio
2. Copia `.env.example` a `.env` y ajusta las variables si es necesario
3. Ejecuta `docker-compose up -d`
4. Accede a `http://localhost:3500`
